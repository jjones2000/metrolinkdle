## Local Development

Run both servers at once:

```bash
./dev.sh
```

Or manually:

### Backend
```bash
cd backend
python -m venv .venv && source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
uvicorn main:app --reload
# http://localhost:8000  |  docs: http://localhost:8000/docs
```

### Frontend
```bash
cd frontend
npm install
npm run dev
# http://localhost:5173  (/api proxied to localhost:8000)
```

## Deploy вЂ” Azure App Service (Backend)

1. Push your code to GitHub

2. In the [Azure Portal](https://portal.azure.com):
   - **Create a resource в†’ Web App**
   - Runtime: **Python 3.11**, OS: **Linux**
   - Plan: **Free (F1)**

3. **Deployment Center в†’ GitHub** вЂ” select your repo, set root directory to `backend`

4. **Configuration в†’ General Settings**, set startup command:
   ```
   uvicorn main:app --host 0.0.0.0 --port 8000
   ```

5. **Configuration в†’ Application Settings**, add:
   ```
   ALLOWED_ORIGINS=https://your-app.netlify.app
   DATA_DIR=/home/data
   ```

6. Your backend URL will be `https://your-app.azurewebsites.net`

---

## Deploy вЂ” Netlify (Frontend)

1. [netlify.com](https://netlify.com) в†’ **Add new site в†’ Import from Git**

2. Set:
   - **Base directory:** `frontend`
   - **Build command:** `npm run build`
   - **Publish directory:** `frontend/dist`

3. **Site configuration в†’ Environment variables**:
   ```
   VITE_API_URL=https://your-app.azurewebsites.net
   ```

4. Deploy вЂ” your app will be live at `your-app.netlify.app`

## API Reference

| Method | Path | Description |
|--------|------|-------------|
| `GET`  | `/health` | Health check |
| `GET`  | `/api/daily` | Today's target stop |
| `GET`  | `/api/daily/{YYYY-MM-DD}` | Target for a specific date (testing) |
| `POST` | `/api/stats` | Record a completed game |
| `GET`  | `/api/stats/{player_id}` | Get stats for a player |
